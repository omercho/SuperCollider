      Description

      See Using Quarks for more information.

      Class Methods

      *gui

      Show GUI to manage quarks.

      *global

      The singleton Quarks instance.

      *checkoutAll

      Check out all quarks (does not install them)

      *listCheckedOut

      List all checked out quarks.

      *listAvailable

      List all available quarks.

      *listInstalled

      List all installed quarks.

      *updateDirectory

      Makes sure you have an up-to-date directory listing.

      *checkout (name, version, sync)

      Check out a specific quark.

      Arguments:

      name

      Name of quark to checkout.

      version

      A Float indicating which version of the quark to checkout.

      sync

      A Boolean

      *install (name, includeDependencies: true, checkoutIfNeeded: true)

      Install a specific quark, by linking it into the user extensions folder.

      Arguments:

      name

      Name of quark.

      includeDependencies

      A Boolean

      checkoutIfNeeded

      A Boolean

      *uninstall (name)

      Uninstall quark.

      Arguments:

      name

      Name of quark.

      *update (quarkName)

      Update quark to latest revision.

      Arguments:

      quarkName

      Name of quark.

      *installed

      Get a list of all installed quarks.

      *isInstalled (name)

      Check if a quark is installed.

      Arguments:

      name

      Name of quark.

      *local

      An instance of LocalQuarks

      *help (name)

      Open help for specific quark.

      Arguments:

      name

      Name of quark.
